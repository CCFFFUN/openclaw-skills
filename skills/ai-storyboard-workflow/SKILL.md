# AI 时代的分镜工作流

过去做分镜的最大成本是"画"——一张张手绘要花几天到几周。AI 工具把这个成本压到接近零，同时显著提升了前期预演的精确度。

## 核心原则

**AI 时代最重要的，不是"让 AI 帮你画分镜"，而是：你是否知道每一个镜头为什么存在。**

分镜不是为了更漂亮，而是为了让故事被真正看见。

---

## 一、短剧剧本写作基本要点

### 1. 开篇 3 秒法则
- 开篇 **3 秒** 必须抛出强冲突或强悬念
- 快速抓住观众注意力
- 适配短视频快节奏观看习惯

### 2. 剧情结构
- **单线紧凑**：不要多线并行
- **反转密集**：每集至少 1—2 个有效反转
- **人物动机简单直白**：不要复杂内心戏
- **台词口语化、短句多**：适配竖屏观看
- **每集结尾埋钩子留悬念**：贴合平台流量逻辑，引导追更

### 3. 平台适配
- 抖音：更强钩子、更快节奏、更短单集
- B站：可容纳更多表达、更长单集、更复杂人物关系
- 快手：偏生活化、接地气、情感共鸣优先

---

## 二、从剧本到分镜脚本

分镜不是复述故事，而是把故事转成 **"观众每一秒看到什么"**。

### 1. 剧本拆分原则

**简单拆分**：
- 把"故事"转化成"画面"
- 明确每一秒观众看到什么、人物做什么、镜头怎么运动、情绪如何变化
- 为后续图像生成、图生视频、剪辑和配音提供依据

**专业拆分**：
- 按 **情绪节拍** 拆分，而不是按自然段落
- 每个分镜单元控制在：
  - `1 个动作`
  - `1 个情绪`

### 2. 拆分步骤

**Step 1｜切时间轴节点**
- 把全集时长切成若干段
- 每段对应一个叙事功能：
  - 钩子
  - 冲突
  - 转折
  - 爆点
  - 悬念收尾

**Step 2｜识别画面锚点**
- 找出剧本中可视化的关键：
  - 动作
  - 表情
  - 道具
  - 环境
- 删除无法被镜头表达的内心独白
- 删除没必要的镜头和叙事

**删除原则**：删除以后故事、镜头还成立，就可以删除。

**Step 3｜逐句拆镜**
- 一句台词或一个动作，对应一个镜头编号
- 标注景别和运镜
- 形成完整分镜表

## 三、5 步 AI 分镜工作流

### Step 1｜剧本输入 AI，生成初版镜头清单

把剧本输入 GPT / Claude / 豆包等模型，要求输出 **初版镜头清单**（Markdown 表格，15—20 镜）。

提示词关键：
- 指定景别
- 指定运动
- 指定时长
- 指定画面内容
- 指定声音信息

### Step 2｜人工审查与修改

AI 生成的镜头清单常见问题：
- 太规整、太平均、太安全
- 节奏平均，没有呼吸感
- 缺少"不归点"或反转镜头
- 情绪推进过平

**人工干预重点**：
- 打破过度均匀的节奏
- 增加有戏的镜头
- 改掉只"说明信息"却不"制造感受"的镜头

### Step 3｜关键镜头生成静态参考图

用 Midjourney / Stable Diffusion / 可灵图像版等工具，为关键镜头生成静态参考图。

**建议做法**：
- 每个关键镜头生成 1 张图
- 贴回分镜表
- 优先覆盖：开场钩子镜头、转折镜头、高潮镜头、结尾呼应镜头

### Step 4｜动态预演（Animatic）

把静态参考图喂给 Runway / Pika / 可灵视频版等工具，生成 **3—5 秒动态片段**，串成 **分镜动画（animatic）**。

**价值**：
- 提前看到节奏是否成立
- 验证镜头衔接是否顺
- 判断情绪是否真的被推起来
- 在拍摄前先发现问题

### Step 5｜拍摄 / 生成执行

**如果最终成片是 AI 生成**：
- 每个镜头的 prompt 要写得高度具体
- 包含镜头类型、运动、时长、光线、色调、风格等信息

**如果最终成片是实拍**：
- 把分镜表 + 静态参考图 + 动态预演统一交给摄影师 / 美术 / 制片

## 四、分镜提示词要素

每个分镜提示词需包含以下要素：

### 必填项
1. **镜头编号**：如 `镜1`、`Shot 01`
2. **景别**：远景 / 全景 / 中景 / 近景 / 特写 / 大特写
3. **角度**：平视 / 俯视 / 仰视 / 倾斜
4. **运动**：固定 / 推 / 拉 / 摇 / 移 / 跟 / 升降 / 环绕
5. **时长**：如 `3s`、`2.5s`
6. **画面描述**：主体、动作、环境、关键细节
7. **声音描述**：对白、音效、音乐建议

### 可选项
8. **转场方式**：硬切 / 叠化 / 跳剪 / 淡入淡出
9. **情绪标签**：如 `紧张`、`温情`、`压抑`
10. **光影提示**：如 `逆光`、`侧光`、`冷暖对比`
11. **色调建议**：如 `低饱和冷蓝`、`暖调胶片感`
12. **参考风格**：如 `《双城之战》风格`、`王家卫色调`

## 五、优势与陷阱

### 优势
1. 成本极低，可反复试错
2. 能提前看到接近成片的画面效果
3. 同一个 prompt 模板可保持视觉连续性
4. 导演、摄影、美术、制片更容易对齐

### 陷阱
1. **画面漂亮但无戏**：AI 容易生成好看画面，但可能缺少戏剧节奏与情绪推进
2. **不同工具风格跳跃**：混用多工具时，若无统一风格策略，会出现视觉断裂
3. **过度对称、过度精致**：AI 倾向于生成"完美"画面，但电影打动人的往往是有生活痕迹、有意外、有不稳定感的画面

### 创作者必须主动校正
- 让镜头有戏，不只是好看
- 让情绪有起伏，不只是美学统一
- 让画面保留"人间烟火气"

---

## 六、实操建议

一个 1 分钟短片的完整 AI 分镜工作流，熟练后可在 **2—4 小时** 内完成。

前提：创作者必须有 **视听语言意识**——知道每个镜头要什么，而不是"让 AI 生成漂亮画面就完事"。
