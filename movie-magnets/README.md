## 使用方法

每个 .magnet 文件是一个完整的磁力链接. 

### 浏览器直接下载
点击 .magnet 文件 → 显示 magnet URI 文本 → 复制全部内容.

### qBittorrent 批量
qBittorrent → "添加 Torrent" → "粘贴 URL" → Ctrl+V 粘贴 magnet 内容 → 确定.

### 一次性全下
查看 _all-magnets.txt (汇总所有12部).
